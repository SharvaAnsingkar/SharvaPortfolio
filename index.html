October 29th, 2023

Renewable Empowerment:
A Solar-Powered Multi-Functional Power Bank for Enhanced Safety and Sustainability in Remote Environments

(PROJECT HAS BEEN FILED AS A PATENT UNDER THE NAME "A SOLAR POWERED MULTI-FUCTIONAL POWER BANK".APPLICATION NUMBER:202421031663 A).
This project focuses on a solar-powered power bank that charges electronic devices using renewable energy. Equipped with a 10,000 mAh battery, solar panel, GPS module, and SIM800L for location tracking, it offers off-grid charging and emergency features. The system includes an inbuilt torch, active buzzer for status alerts, and an SOS function for safety. Designed for outdoor activities, this eco-friendly device promotes sustainability, reliability, and safety, providing users with a versatile solution for emergencies and recreational use.

June 10th, 2023

Traffic Congestion Control System

(RESEARCH PAPER PUBLISHED AND AWARDED BEST RESEARCH PAPER AWARD IN INTERNATIONAL CONFERENCE ON AIML HELD AT VIGNAN INSTITUTE OF TECHNOLOGY, HYDERABAD, TELANGANA, INDIA).
This project aims to reduce traffic congestion by dynamically adjusting red traffic signal durations based on real-time traffic data. By analyzing vehicle volumes on weekdays and weekends, the system tailors signal timings to optimize traffic flow, reducing wait times and congestion. Additionally, strategically placed sensors detect approaching vehicles, sending alerts to traffic authorities when no movement is detected, signaling potential blockages. This proactive approach allows for quick route diversions, preventing further buildup. The automation of these processes ensures a responsive, efficient traffic management system without manual intervention.

November 18, 2024

Solspot:
Efficient Parallel Parking

This project introduces a smart and efficient solution to tackle parking challenges in congested areas. Utilizing components such as a solar panel, ESP32 microcontroller, L298N motor driver, and UBEC, the system enables precise control of vehicle movement in all directions—forward, backward, left, and right. With Wi-Fi connectivity, users can seamlessly control the car, enhancing maneuverability and enabling smooth parallel parking. By integrating technology-driven solutions, this project reduces parking time and provides a foundation for advanced parking systems, aligning with ongoing innovations like 360-degree turning vehicles and automated parking systems.

June 2nd, 2023

Design and Fabrication of Blink Detection Communication Device for Hearing and Speech Impaired Person

This project introduces a communication device for individuals with hearing and speech impairments by leveraging blink detection technology. Integrated with IR sensors in glasses frames, an Arduino Uno microcontroller, and an LCD screen, the device detects eye blinks and displays corresponding messages in real-time. This innovative solution provides seamless, instant communication, addressing the need for assistive technology that caters to multiple impairments simultaneously.

November 17, 2024

AutoValue
Car Resale Price Prediction System

A machine learning-based system to predict car resale values using features like make, model, mileage, year, fuel type, and prior usage. Advanced data preprocessing handles missing values, normalizes features, and encodes categorical data. Multiple regression models—including linear regression, decision trees, and ensemble methods—are explored and optimized. Validated on real-world data, the system ensures high accuracy and generalizability, supporting efficient pricing and decision-making in the used car market.

November 14, 2024

ESP32 Wireless FTP

A system that implements a wireless FTP server on the ESP32 microcontroller, enabling bidirectional file transfers between an SD card and a smartphone without internet connectivity. The system allows both file retrieval and data upload, functioning like an external hard drive accessible via Wi-Fi. Using the AndFTP app on the smartphone, users can interact with the ESP32 server. The system's performance includes transfer speeds of up to 5 Mbps, with normal speeds observed for larger files like photos.
